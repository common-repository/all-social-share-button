== Description ==

This a very simple plugin for WordPress. What it does is add easy social networking sharing button for each of your blog posts. Share by Email, Twitter, Facebook, Digg and by most of the social websites.
We have added the beautiful buttons in form of icons. these are small and effective and responsive too.

== Installation ==

1. Upload `all-social-share-button` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. click on admin menu option(All social share button)
4. Add title in the input box
5. select the site names, to share youe post url
6. In the last you just have to press save button.
